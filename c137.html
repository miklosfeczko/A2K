Updated
  • 2019-11-25

  • I decided to hurry things up a bit so I ended up reusing code from my previous projects so I did not have to start everything from scratch.


    After making the basic API calls I headed into creating the UI of the landing page. My idea was to create something unique yet simple so I put the navigation elements on the sides... vertically.


    Then I dived into the application part where I used a simple two button layout to browse between pages with the same navigation.


    The last thing was to expand the functionality with JavaScript and to actually make the buttons working. I finished the project, the code could use some sort of refactoring and maybe I will polish it a bit but the app itself is in working condition right now.


    I am glad I have particapted in this challenge also I had a lot of fun making this project.
